## how to use

- run composer install 
- create database
- copy .env.example to .env
- enter database name , use and password in .env file
- run `php artisan key:generate`
- run `php artisan migrate --seed`
- run `php artisan jwt:token`
- run `php artisan serve`

you are ready to go
